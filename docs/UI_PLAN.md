# UI 规划

## 信息架构

主导航：

- 书架：阅读入口、进度、分类筛选、本地导入
- 发现：书源搜索、搜索结果、加入书架
- NAS：连接管理、目录浏览、文件导入
- 下载：队列、进度、失败重试、缓存管理
- 设置：阅读样式、书源、替换净化、同步、备份

## iPhone

使用底部 Tab 承载五个主区。阅读器全屏，点击页面中心呼出顶部和底部控制栏。

阅读页控制：

- 顶部：返回、书名、更多
- 底部或工具栏：上一章、下一章、目录、样式、进度
- 样式面板：字号、行距、段距、加粗、主题

## iPadOS

使用 `NavigationSplitView`：

- 左栏：主导航
- 中栏：书架/文件/结果列表
- 右栏：详情或阅读器

阅读器支持宽屏双页阅读、键盘翻页、外接键盘快捷键。

## macOS

macOS 以资料库工具体验为主：

- 左侧固定导航
- 多窗口阅读
- 菜单栏命令
- 拖拽导入 TXT/EPUB
- 右键菜单管理缓存与书籍

## 当前可用界面

- 书架支持统计概览、网格展示、本地 TXT/EPUB 导入、书名作者搜索、正文命中展示。
- 书籍详情页支持阅读入口、目录跳转、进度/来源/章节统计和删除。
- 发现支持书源搜索、结果分层展示、加入书架、直接下载。
- NAS 支持手动连接、协议标签、Bonjour 发现、WebDAV 子目录浏览和文件导入。
- 下载页以任务看板形式展示状态，并支持暂停、继续、重试、清理完成任务。
- 设置页支持阅读样式、书源启停、新增书源、导入 JSON、书库备份和恢复。

## 下一轮 UI

- 书籍详情页继续补来源详情、重新下载入口。
- EPUB 封面图片渲染到书架和详情页。
- 下载和 NAS 的批量选择模式。

## 视觉原则

- 阅读页安静，内容优先。
- 工具页信息密度高，便于扫描。
- 书架保留封面感，但不做商业书城风格。
- 避免大面积单色主题，默认纸页色 + 深灰文字。
